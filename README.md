# <center>路径结构</center>

## <center>\\.about</center>

\\.about 内容中主要放置的是：

1. 该项目的背景介绍。
2. 该项目的主要内容介绍。
3. 项目作者信息与联系方式以及转载须附内容。

## <center>\\.content</center>

\\.about 内容中主要放置的是：

1. 该项目的主要内容

    

## <center>\\.others</center>

\\.others 内容中主要放置的是:

1. 项目完成度
2. 完成后的感想总结
3. 对项目未来改进提出的方向和建议
